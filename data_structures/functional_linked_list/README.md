# Functional Linked List

## About

This is a functional singly linked list implementation in Javascript.

It implements the following functionality:

```
isEmpty
length
reduce
reduceRight
map
```

## Getting Started

To install all dependencies:

```bash
npm install
```

To run all tests:

```bash
npm test
```
